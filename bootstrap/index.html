CSS Box Model

The Box Model is essentially a box which wraps around every HTML element. This consists of:
margins, borders, padding and the actual content. The Margin is a clear box around the border
of the content. The Margin is a specific area that has been coded. It is in between the browser
and the border. It gives the content enough space from the edge of the web page. The Border
has different values which is used to design the element. The border is used to make the
content stand out more. This can be done by changing the thickness, style or colour. The
Padding gives space between the border and the content. The Padding is transparent. It
clears an area around the content. The Content is what is actually on the webpage.This can
vary from page to page, it is up to the designer what it looks like. This is done using CSS and
HTML.

CSS Models

There are different CSS styles when creating your own web page with HTML and CSS.
The three different CSS types are; Inline, Internal and External.Inline is a style which
can be used on any web page as you are using a specific tag. The HTML tag which
you will use will begin with style, this is an attribute. By using Inline you will be able
to type anything you want. Inline is quick and simple to use if you know how to use it.
This is because you can simply add or take away anything without having to make
different files for every change you make. An advantage of using Inline is that there is
fewer HTTP requests. This means that the web page will load quicker and be more
responsive than it would be if Internal or External was used. Another advantage is that
Inline CSS can be used to debug other pages when an error is found. This means that
it is quicker for the error to be fixed. Inline also allows for more creativity and freedom
when designing a web page with HTML. A disadvantage is that you will have to manually
insert all of the HTML. This means it will take a long time. It also means that if an error is
made you will have to erase it and redo it again. Another disadvantage is that if you
wanted to change something, such as the colour, then you will have to add an inline style
to each p tag. This can take a while if there is a lot of content and it means they will have
to add it to each tag.Internal Style is also known as Embedded Style. Internal is used to
create a whole page within a website. This is done in a separate file just like External is.
An advantage of Internal Style is that it allows the designers to keep a same style on every
part of the file they are creating. This means that while the user is creating the page, they
won’t have to apply or type out the same HTML code for every element they use. For
example, if the creator wanted all of the heading to be the same, they would choose their
font such as “Georgia”, then they would just need to add an inline attribute p which goes
inside the internal document for that specific section. A disadvantage of using Internal
Style is that it takes a while to load the page. This is because there is not a lot of HTTP
demands. This makes Internal Style slower than both Inline and External. Another
disadvantage is that once the file is created and saved, that file can only be used if the
creator has inserted to use the HTML code which is going to be used for the web page. If
they wish to edit the file again, they would need to create a new file and copy everything
over again. External Style is a file which will only contain HTML coded within it and will
need to be saved as ‘CSS’. With Inline, the element style would be used, however with
External the element link will be used. This means that the file style sheet will be linked to
the page or pages of the website. This means that when using External the rule will only
need to be set once. An advantage of External Style is that it enables the designer to have
complete access of the page structure. It will also separate it from your website and content.
When using External Style it means that the designer can reuse the HTML code if it has been
kept in a separate file, this means that the whole code won’t need to be rewritten again. A
disadvantage of using External Style is that you won’t be able to access the website if the
external file is missing, which can cause a lot of problems. If a large file is being used to insert
into the website then it can cause the download time to be very slow when trying to open it
on the internet.
